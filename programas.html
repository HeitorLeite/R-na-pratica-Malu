Exemplo: Cálculo de Fatorial e Verificação de Número Primo

Este exemplo mostra como calcular o fatorial de um número e verificar se ele é primo:

# Função para calcular o fatorial
fatorial <- function(n) {
  if(n == 0) {
    return(1)
  } else {
    return(n * fatorial(n - 1))
  }
}

# Função para verificar se um número é primo
eh_primo <- function(n) {
  if(n <= 1) {
    return(FALSE)
  }
  for(i in 2:sqrt(n)) {
    if(n %% i == 0) {
      return(FALSE)
    }
  }
  return(TRUE)
}

# Testando as funções
numero <- 5
cat("O fatorial de", numero, "é", fatorial(numero), "\n")
if(eh_primo(numero)) {
  cat(numero, "é um número primo.\n")
} else {
  cat(numero, "não é um número primo.\n")
}
