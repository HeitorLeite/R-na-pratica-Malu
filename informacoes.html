Vídeo

Origem da Linguagem R

A linguagem R foi desenvolvida por Ross Ihaka e Robert Gentleman na Universidade de Auckland, Nova Zelândia, no início dos anos 90. Inspirada na linguagem S, R tornou-se amplamente utilizada para análise estatística e ciência de dados, evoluindo com contribuições de uma vasta comunidade global.

Fundamentos da Sintaxe

A sintaxe de R é intuitiva e acessível para iniciantes. Exemplos básicos incluem:

  • Atribuição: x <- 10
  • Operações Matemáticas: total <- x + 5
  • Criar Vetores: numeros <- c(1, 2, 3, 4, 5)
  • Funções: media <- mean(numeros)

Essa linguagem flexível é poderosa para manipulação de dados e criação de visualizações complexas.

Variáveis

Em R, as variáveis são utilizadas para armazenar dados que podem ser utilizados e manipulados posteriormente. Exemplos incluem:

  • Números: x <- 42
  • Textos: nome <- "R Linguagem"
  • Vetores: idades <- c(25, 30, 35, 40)
  • Listas: info <- list(nome="Ana", idade=28)

Funções

Funções em R são blocos de código reutilizáveis que executam tarefas específicas. Você pode usar funções integradas ou criar suas próprias:

  • Funções Integradas: sqrt(16) (calcula a raiz quadrada de 16)
  • Definindo Funções:
minha_funcao <- function(a, b) {
    return(a + b)
}
resultado <- minha_funcao(5, 3)

Bibliotecas

Bibliotecas em R são coleções de funções e dados pré-compilados que você pode usar para estender as capacidades do R. Para usar uma biblioteca, você precisa carregá-la:

  • Carregar Biblioteca: library(ggplot2)
  • Instalar Biblioteca: install.packages("ggplot2")

Pacotes Populares

R possui uma rica coleção de pacotes que ampliam suas funcionalidades. Alguns dos mais utilizados incluem:

  • dplyr: Ferramentas para manipulação de dados.
  • ggplot2: Criação de gráficos sofisticados.
  • tidyr: Organização de dados em formatos estruturados.
  • shiny: Desenvolvimento de aplicativos web interativos.
  • caret: Funções para aprendizado de máquina.

Esses pacotes são apenas uma amostra do vasto ecossistema de R, que continua a crescer com contribuições da comunidade.

Estrutura da Linguagem R

A linguagem R possui uma estrutura organizada em diversos componentes essenciais:

  • Ambiente: R é um ambiente interativo onde você pode executar código e visualizar resultados imediatamente.
  • Objetos: Tudo em R é um objeto, incluindo números, vetores, funções e modelos.
  • Vetores e Matrizes: Estruturas de dados fundamentais que permitem manipulações e cálculos eficientes.
  • Data Frames: Estruturas de dados bidimensionais, semelhantes a tabelas, ideais para dados estatísticos.
  • Listas: Coleções de objetos que podem ser de diferentes tipos e tamanhos.
